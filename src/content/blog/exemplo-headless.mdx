# Este é um Post Headless

Este post está configurado com `showMetadata: false` e `headless: true`, o que significa:

- Não exibe a seção de metadata no layout
- Não oculta o título H1 do MDX (porque não há duplicação)

## Vantagens do Modo Headless

- Controle total sobre o layout diretamente no MDX
- Ideal para conteúdo personalizado
- Sem duplicação de informações

## Quando Usar

Use o modo headless quando:
- Quiser controle total sobre a apresentação
- Não precisar da metadata padrão
- Tiver um design específico em mente
