# GFM + shadcn: mapeamento 1:1

Este documento exercita, usando apenas GitHub Flavored Markdown (GFM), todos os elementos mapeados em `mdx-components.tsx`. Onde faz sentido, o render usa componentes shadcn/ui equivalentes (blockquote → Alert, hr → Separator, tables → Table, task lists → Checkbox).

## Tabelas (GFM)

| Componente | Mapeado para | Observação                  |
|------------|--------------|-----------------------------|
| hr         | Separator    | via elemento `---`          |
| blockquote | Alert        | com AlertDescription interno |
| table      | Table*       | header, body, row, head, cell |
| task-list  | Checkbox     | checkboxes somente leitura   |

> * O Markdown de tabela é mapeado para os primitvos shadcn de tabela.
